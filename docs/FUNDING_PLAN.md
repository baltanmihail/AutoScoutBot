## 2. Стратегия: УМНИК + Студенческий стартап

### 2.1 Программа «УМНИК» (Фонд содействия инновациям)

**Что это:** Грант до 500 тыс. руб. на 2 года для молодых исследователей (18-30 лет).

**Что нужно подготовить:**
- [ ] Заявка на сайте fasie.ru (раздел «УМНИК»)
- [ ] Описание научно-технической разработки (наш AI-скоринг)
- [ ] Обоснование научной новизны (нет аналогов AI-скаутинга в РФ)
- [ ] План работ на 2 года
- [ ] Рабочий прототип (есть — Telegram-бот + ML pipeline)
- [ ] Презентация 10-15 слайдов
- [ ] Отзыв научного руководителя

**Научная новизна (ключевые тезисы):**
1. Первая в России автоматическая система скаутинга стартапов с ML-скорингом
2. Мультимодальная модель: табличные данные (35 признаков) + текстовые embeddings
3. Перекрёстная верификация из государственных источников (ЕГРЮЛ, ФИПС)
4. SHAP-объяснимость для прозрачности AI-решений
5. Обучение на уникальном датасете 5000+ стартапов Сколково

## 3. Необходимые материалы

### 3.1 Научная публикация

**Рекомендуемые площадки:**
- **Конференция АИСТ** (Анализ Изображений, Социальных сетей и Текстов) — профильная для NLP/ML
- **Конференция «Ломоносов»** (секция ИИ) — для студентов МГУ и партнёров
- **Журнал «Программирование»** (ВАК, Scopus) — для серьёзной публикации
- **CEUR Workshop Proceedings** — быстрая публикация на англ.

**Примерная структура статьи:**
1. Introduction: проблема ручного скаутинга
2. Related Work: Crunchbase, СПАРК, Сбер Юнити — что есть и чего нет
3. Method: 35-dim features + LightGBM + SHAP + external data
4. Dataset: 5166 стартапов Сколково
5. Experiments: K-fold CV, MAE, RMSE, R², сравнение с baseline
6. Results: метрики модели + примеры SHAP-объяснений
7. Conclusion + Future Work

### 3.2 Метрики для заявки

**Что нужно измерить и зафиксировать:**
- [ ] MAE, RMSE, R² модели на кросс-валидации
- [ ] Сравнение ML-модели vs старых эвристик (DeepTech/TrafficLight)
- [ ] Время inference (< 100 мс)
- [ ] Покрытие данных: % стартапов с данными ЕГРЮЛ, финансами
- [ ] User satisfaction (опрос пользователей бота)

### 3.3 Демо и презентация

- [ ] Видеодемо работы бота (2-3 мин)
- [ ] Скриншоты: поиск, анализ, SHAP-объяснение
- [ ] Архитектурная диаграмма
- [ ] График: точность модели по эпохам
- [ ] Сравнительная таблица с конкурентами
