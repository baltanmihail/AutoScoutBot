## 2. Себестоимость одного запроса

### 2.1 Профиль типичного запроса

| Параметр | Обычный запрос | Глубокий анализ |
|----------|---------------|-----------------|
| Input токены | ~2 500 | ~4 000 |
| Output токены (обычная модель) | ~800 | ~2 000 |
| Output токены (thinking-модель) | ~2 000 * | ~3 500 * |

> \* Thinking-модели генерируют скрытую цепочку рассуждений, что увеличивает  
> реальный output в 2,5-3 раза от видимого ответа пользователю.

### 2.2 Расчёт себестоимости

#### Обычный запрос (поиск + рекомендация)


## 8. Риски и рекомендации

### 8.1 Риски

| Риск | Вероятность | Влияние | Митигация |
|------|------------|---------|-----------|
| Рост цен NeuroAPI | Средняя | Высокое | Мониторинг; возможность смены провайдера |
| Низкая конверсия (<10%) | Средняя | Среднее | Улучшение UX, больше бесплатных фич |
| Исчерпание GigaChat токенов | Высокая | Низкое | Переключить _internal на DeepSeek (~25₽/1M) |
| Thinking-модели дорожают | Низкая | Высокое | Кэширование ответов, лимиты на thinking |

### 8.2 Рекомендации

1. **Начать с 500-1000₽ на NeuroAPI** для тестирования (хватит на ~500-1000 запросов Gemini 3 Pro)
2. **Мониторить ARPU** -- если пользователи предпочитают Gemini, маржа будет ниже
3. **Стимулировать Opus 4.6** -- лучшая маржа (×3,6). Добавить "попробуй 💎 Claude Opus" промо
4. **Кэширование**: для повторяющихся запросов кэшировать ответы LLM (экономия ~20-30% API)
5. **Когда GigaChat кончится**: переключить `_internal` на DeepSeek V3 через NeuroAPI
   (стоимость: ~25₽ input + ~37₽ output за 1M → ~0.06₽/запрос, пренебрежимо мало)

## Приложение: формулы

```python
# Себестоимость 1 запроса
cost = (input_tokens / 1_000_000) * price_input + (output_tokens / 1_000_000) * price_output

# Для thinking-моделей: реальный output ≈ visible_output × 2.5
thinking_output = visible_output * 2.5

# ARPU (Average Revenue Per User)
arpu = sum(tier_requests * tier_price for each tier) / paying_users

# Маржа
margin = (revenue - api_cost - infra_cost) / revenue × 100%

# Точка безубыточности (в пользователях)
breakeven = fixed_costs / (arpu × conversion_rate)
```
